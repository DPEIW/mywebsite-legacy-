Formula SAE / PCB

Designed and iterated a pitot tube data acquisition PCB for Wisconsin Racing's aerodynamics system. The board reads airspeed sensor data, conditions the signal, and interfaces with an ESP32 so the team can debug and validate vehicle aero behavior more efficiently.
